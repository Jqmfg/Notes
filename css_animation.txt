http://daneden.github.io/animate.css/
This has css animation resources
https://github.com/daneden/animate.css
This is the github page for more info on how to use it
To use this css package, you first must link the stylesheet with
  <head>
    <link rel="stylesheet" href="animate.min.css">
  </head>
This will add all of the classes to your html file
After that you must add in the animated class in order to tell it that you can animate it
You can also add the infinite class so it loops
After that you must add one of the following classes for the animation itself
